# AperoPHP - CakePHP

## Installation

### Avec docker

```bash
./install.sh
```

- App : http://localhost:8080
- PhpMyAdmin : http://localhost:8081

### Sans docker

Requis :

- PHP >= 5.6
- Composer
- L'extension PHP mbstring
- L'extension PHP intl
- L'extension PHP simplexml

```shell
composer install
```
